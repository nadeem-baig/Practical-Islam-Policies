Data Deletion Request Policy

Apps that access user data must provide a way for users to request that their data be deleted. This is an essential part of respecting user privacy and ensuring data security.

How to Request Data Deletion

To comply with this policy, our app provides the following methods for users to request data deletion:

  • A data deletion request callback feature within the app.
  • Detailed instructions on our website about how users can request data deletion.

When a data deletion request is received, it is processed promptly. Our team ensures that all personal data related to the requesting user is securely and completely removed from our servers and databases.

Frequently Asked Questions

FAQs:

  • How long does it take to process a data deletion request?
    Data deletion requests are typically processed within 48 hours of receipt.
  • Will I receive confirmation once my data is deleted?
    Yes, we send a confirmation email once the data deletion process is complete.
  • Can I request to view my data before it is deleted?
    Absolutely. You can request to view your data as part of the data deletion process.
